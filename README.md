<h1>Weather app for TOP</h1>
Allows you to enter a city, which then displays information such as temperature, description, precipitation, and a 7-day forecast.</br>
Also allows you to swtich between Farhenheit and Celcius.</br>
Accesses Visual Crossing's weather API.
